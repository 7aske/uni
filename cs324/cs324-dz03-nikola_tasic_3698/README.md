# CS324 DZ03

## Zadatak #1

Napisati program u kojem ubacujete broj trenutno polozenih ispita. Ulazne podatke sačuvati
u imenik koji sadrži šifru predmeta kao ključ predmeta, a vrednost jeste ocena na položenom
predmetu.
Zatim, izvršiti upit za predmet po šifru predmeta.
Izbaciti kao izlaz da li je predmet položen ili ne, i ako jeste, sa kojom ocenom.

## Zadatak #2

Napisati program koji će generisati n nasumičnih celih brojeva, gde je n broj indeksa studenta,
i smestiti u listu, i sortirati listu u padajućem redosledu. Koristiti import random za dobijanje
nasumičnih brojeva.
